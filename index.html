Pool Info

Shamrock Pool uses cloud based servers to enable elasticity, scaling and 99.9% uptime.

The pool is set up with multiple relay and core nodes distributed across the globe with a dedicated cloud server for each one. Each server runs on 8GB RAM, 4vCPUs, 4TB of transfer with 1Gbps bandwidth (around 125MB/s), and 80GB SSD storage.

The cloud-based infrastructure allows us to scale our pool as our needs grow.
